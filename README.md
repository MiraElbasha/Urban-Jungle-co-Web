# Urban Jungle Co. Website

**Author:** Mira Elbasha

---

## Project Overview

**Urban Jungle Co.** is a **front-end skills showcase website** built with **HTML, CSS, and JavaScript**. This project demonstrates my ability to create a visually appealing and interactive website to display products and key sections commonly found in modern websites.

> ⚠️ This is **not an e-commerce website**. There is no actual purchasing functionality. It is purely for demonstrating front-end development skills.

---

## Features

- **Responsive Layout** – Works on desktop, tablet, and mobile using CSS media queries and Bootstrap grid.  
- **Sticky Navbar** – Navigation bar changes style on scroll.  
- **Mobile Menu Toggle** – Collapsible navigation menu for smaller screens.  
- **Smooth Scroll** – Scrolls smoothly to sections when navigation links are clicked.  
- **Products Display** – Trending and popular products with pagination and slider functionality.  
- **Cart UI Simulation** – Clicking products increments a cart counter (stored in `localStorage`) to demonstrate dynamic front-end behavior.  
- **Scroll Reveal Animation** – Elements fade in as they appear in the viewport.  
- **Flash Sale Countdown** – Dynamic countdown timers for special offers.  
- **Back to Top Button** – Quickly scroll back to the top of the page.  
- **Customer Testimonials Section** – Cards displaying client opinions with avatar images.

---

## Technologies Used

- **HTML5** – Semantic markup for website structure.  
- **CSS3** – Styling, responsive design, and animations.  
- **JavaScript (ES6)** – DOM manipulation, event handling, cart simulation, sliders, countdown timers.  
- **Bootstrap 5** – Grid system and responsive utilities.  
- **Font Awesome** – Icons for social links, cart, and features.  
- **Google Fonts** – Custom typography using Instrument Sans.

## How to Run Locally

1. Clone or download the repository.  
2. Make sure all **images are inside the `images` folder**.  
3. Open `index.html` in a browser.  
4. For best results, use **Live Server** in VS Code to view interactive features.
